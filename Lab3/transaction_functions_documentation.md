# ❓ Контрольные вопросы

## 1. Какие методы массивов используются для обработки объектов?
Чаще всего используются:
- `map()` — для преобразования данных
- `filter()` — для отбора элементов по условию
- `reduce()` — для получения одного результата из массива
- `find()` — для поиска одного элемента
- `forEach()` — для перебора без создания нового массива

---

## 2. Как сравнивать даты в строковом формате?
Если даты записаны в формате `YYYY-MM-DD`, их можно сравнивать как строки, потому что порядок символов совпадает с хронологическим порядком.

Более надёжный способ — преобразование в объект Date:
```js
new Date(date1) < new Date(date2)
```

---

## 3. Разница между map(), filter(), reduce()

- `map()` — изменяет каждый элемент массива и возвращает новый массив
- `filter()` — оставляет только элементы, которые подходят под условие
- `reduce()` — сводит весь массив к одному значению (например сумма или объект)

Пример логики:
- map → преобразовать
- filter → отобрать
- reduce → посчитать итог
